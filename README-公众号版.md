# 微信单词机器人 · 公众号版使用说明

本包是**公众号接入版本**，不含个人微信桥接（cc-weixin）。适合想用**微信公众号**来提供查词 / 复习 / 测验 / 学习统计的用户。

> 与个人微信版的区别：个人微信版靠扫码登录 `cc-weixin` 桥接；本版把机器人挂到你自己的公众号「服务器配置」上，用户给公众号发消息即触发。两种接入二选一，代码共用。

---

## 一、准备

1. 一个微信公众号：**接口测试号**（开发最方便、免认证，推荐先试）／**订阅号**／**服务号**均可。
2. 一台常驻运行机器人的机器（本机、云服务器都行）。
3. 一个**内网穿透**工具把本地 9090 端口暴露到公网：
   - 国内推荐 **natapp**（免费隧道即可，稳定）；
   - 海外可用 **ngrok**（免费版域名每次变）。

> ⚠️ 能力差异：未认证订阅号 **没有客服消息权限**，因此「每日复习推送 / 周报」会自动停用，但**查词、测验、学习统计（发「统计」「进度」「我学了啥」）完全正常**。要解锁推送需认证服务号。

---

## 二、快速开始（5 步）

```bash
# 1. 安装 Python 依赖
pip install -r requirements.txt

# 2. 配置 .env（见下方字段说明）
cp .env.example .env
#   然后编辑 .env，至少填好 MP_APPID / MP_APPSECRET / MP_TOKEN，
#   并设 WECHAT_CHANNEL=official

# 3. 启动机器人（默认监听 9090）
python3 main.py

# 4. 另开终端，启动内网穿透，把 /wechat 暴露为公网
#    natapp 示例（隧道指向本地 9090）：
#    前台会显示 http://xxxx.natappfree.cc
#    你的回调地址就是  http://xxxx.natappfree.cc/wechat

# 5. 去公众号后台填「服务器配置」（见第三节），提交成功即可
```

## 三、公众号后台配置

以**接口测试号**为例（订阅号/服务号步骤一致）：

1. 打开 https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login ，微信扫码登录，拿到 `appID` 与 `appsecret`。
2. 在「接口配置信息」填写：

| 字段 | 填什么 |
|------|--------|
| **URL（服务器地址）** | `你的公网地址/wechat`（例：`http://xxxx.natappfree.cc/wechat`） |
| **Token（令牌）** | 与你 `.env` 里的 `MP_TOKEN` **完全一致**的任意字符串 |

3. **消息加解密方式** 选 **明文模式**（最简单，无需 EncodingAESKey；EncodingAESKey 随便点「随机生成」即可）。
4. 填好 `appID` / `appsecret` 到 `.env` 后点「提交」，**微信会立刻发一次 GET 验证**，显示「配置成功」即握手完成。

> URL 的**路径必须是 `/wechat`**；Token 必须与 `MP_TOKEN` 一致，否则验证失败。

---

## 四、.env 关键字段

| 字段 | 说明 |
|------|------|
| `WECHAT_CHANNEL` | 填 `official`（公众号模式） |
| `MP_APPID` | 公众号 appID |
| `MP_APPSECRET` | 公众号 appsecret |
| `MP_TOKEN` | 你自己定的令牌，须与后台 Token 一致 |
| `MP_CALLBACK_PATH` | 回调路径，默认 `/wechat`（一般不用改） |
| `PORT` | 本地端口，默认 `9090` |

---

## 五、验证（端到端）

1. 机器人 + 穿透已启动，后台「配置成功」。
2. 微信里关注你的测试号 / 公众号。
3. 发 `hello` → 收到音标 / 释义 / 例句。
4. 发 `苹果` → 走中译英查词返回英文卡片。
5. 发 `统计`（或 `进度`、`我学了啥`）→ 收到学习统计。
6. 慢查询时先看到「⏳ 正在查询」，随后由客服消息补推结果（需客服消息权限）。

---

## 六、常见问题

| 现象 | 原因 / 解决 |
|------|------------|
| 提交 URL 提示「验证失败」 | Token 与 `.env` 不一致；或本地未启动 / 穿透未通 / 路径非 `/wechat` |
| 配置成功但收不到回复 | 看启动日志 `[wechat]` 有无异常；确认 `WECHAT_CHANNEL=official` |
| 慢查询后无补推 | 未认证订阅号无客服消息权限（正常现象）；或服务号时 `MP_APPID/MP_APPSECRET` 错误 |
| 想切回个人微信 | `.env` 设 `WECHAT_CHANNEL=personal`，改用个人微信版包 |

---

## 七、安全提醒

- `.env` 含密钥，**切勿提交到任何公开仓库**。本包已剔除所有密钥与日志。
- 更完整的架构与云服务器（Nginx + HTTPS）部署，见同包内的 `WECHAT_OFFICIAL_SETUP.md`。
